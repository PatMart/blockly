Using Blockly for the Hour of Code

To run Blockly on a computer when not connected to the Internet:
  1. Download blockly.zip while connected to the Internet.
  2. Transfer the file to any other computers and unzip it.
  3. Open "blockly-zip/index.html" in a modern web browser.
  4. Select your preferred language from the dropdown menu at the top right of the screen.
Languages completely translated: Dansk, Deutsch, Ελληνικά, Español, فارسی, Français, Magyar, Íslenska, Italiano, Nederlands, Română, Português Brasileiro, Русский, Svenska, Українська, Tiếng Việt, 簡體中文, 正體中文.
Languages partially translated (maze tutorial): العربية, Brezhoneg, Català, Česky, Euskara, Galego, Interlingua, 日本語, 한국어, Latviešu, Македонски, Bahasa Melayu, Polski, Piemontèis, Slovenčina, Српски, Svenska, Kishwahili, ภาษาไทย, Türkçe.
